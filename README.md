# Описание скриптов 

## nmap_hostup_and_enum4linux-ng

  Данный скрипт предназначен для автоматизированной инвентаризации сетевой системы в заданной подсети. Он принимает на вход диапазон IP-адресов в формате CIDR и выполняет первичное обнаружение активных узлов с использованием утилиты nmap. На основании результатов сканирования формируется список доступных хостов, который используется для дальнейшего анализа.

  Для каждого обнаруженного активного хоста скрипт запускает инструмент enum4linux-ng, выполняя сбор информации о доступности сервисов LDAP, LDAPS и SMB. Результаты анализа сохраняются в отдельные файлы, что обеспечивает удобство последующего изучения и документирования. В завершающей части работы скрипт автоматически анализирует полученные данные и выводит список узлов, на которых обнаружены открытые порты 389, 636, 445 или 139, позволяя оперативно выявить потенциально критичные сетевые сервисы.
  
  Скрипт может использоваться в учебных целях, при проведении лабораторных работ, а также в рамках предварительного анализа защищённости локальных сетей.

  **На вход скрипту передаётся один параметр — диапазон сетевых адресов в формате CIDR:** xxx.xxx.xxx.xxx/mm, где xxx.xxx.xxx.xxx - базовый IP-адрес сети; mm - длина маски подсети. На пример : 192.168.10.0/24
  
  **В результате работы скрипта формируются следующие выходные данные:**
  - Файл со списком активных хостов, обнаруженных в заданной подсети: nmap_scan_hostup_xxx.xxx.xxx.xxx_mm.txt
  - Каталог с результатами анализа enum4linux-ng: enum4linux_scan_hostup_xxx.xxx.xxx.xxx_mm/
  - В каталоге для каждого активного хоста создаётся отдельный файл: enum4linux_scan_hostup_xxx.xxx.xxx.xxx.txt
  - Консольный отчёт, выводимый по завершении работы скрипта, содержащий список IP-адресов хостов, на которых обнаружены один или несколько открытых портов 389, 636, 445 или 139.

  **Установка**

  ```bash
    git clone https://github.com/RomanovYury/script
    cd ./sctipt
    chmod +x nmap_hostup_and_enum4linux-ng.sh 
  ```
  
  Для работы скирпта ему нужны следующие пакеты:
  - [nmap](https://www.kali.org/tools/nmap/)
  - [enum4linux-ng](https://www.kali.org/tools/enum4linux-ng/)

  **Запуск скрипта**
  ``` bash
  ./nmap_hostup_and_enum4linux-ng.sh xxx.xxx.xxx.xxx/mm #где xxx.xxx.xxx.xxx - базовый IP-адрес сети; mm - длина маски подсети. На пример : 192.168.10.0/24
  ```
